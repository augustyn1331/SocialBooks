# SocialBooks

### Installation

1. Clone the repo locally:

```sh
https://github.com/augustyn1331/SocialBooks.git
```

2. Navigate into project
```sh
cd SocialBooks
```

3. Install dependencies
```sh
npm install
```

4. Configurate iOS project
```sh
cd ios
pod install
```

5. Go back to main project folder
```sh
cd ..
```

### Run application

##### iOS #####
```sh
npm run ios
```

##### Android #####

```sh
npm run android
```
